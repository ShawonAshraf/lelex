Visit the [Releases](https://github.com/ShawonAshraf/lelex/releases) section of the repo.

## Building on your machine
- Clone the repo
- `cd` into the directory
- Run the `npm` `scripts`

```bash
npm install
npm start
```

- If you want to make modifications and want a file watcher
```bash
npm run watch
```

## Packaging
